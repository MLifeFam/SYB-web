## ❓ WHY

기존 세종대학교 홈페이지는 **학과정보를 찾기에 굉장히 불편**했습니다. 통합검색을 지원하지만 검색 기능 자체가 제대로 동작하지도 않고, 검색 결과 마저도 가독성이 안좋아서 학생들은 일일이 사이트를 뒤져가면서 정보를 찾아야했습니다. <br/>

학과 정보를 찾지 못한 학생들은 자연스럽게 학과 사무실의 조교들에게 전화,메일,방문을 하여 문의를 하는일이 잦았고, 분명히 홈페이지에 올라와있는 학과 정보임에도 이러한 비효율적인 과정이 반복되었습니다. <br/>
    
따라서 **학부생에게는 학과 정보를 쉽게 찾을 수 있도록, 조교들에게는 불필요한 업무 부담감을 줄여주기 위해** 세종대 소융봇이라는 카카오톡 기반 챗봇 서비스를 개발하게 되었습니다.
    
<br/>
  
## 🎙 Function

### 챗봇 내 기능

- 학과별 학과공지, 학과별 교과과정, 학과별 사무실 정보, 수강편람, 학사 일정을 버튼 클릭으로 손쉽게 찾을 수 있습니다
- 교수님 성함 검색을 통한 교수님 연구실 정보, 강의실 검색을 통한 강의실 시간표를 제공합니다
- 자유롭게 질문을 검색하면 머신러닝을 통한 문장인식 과정을 거쳐 조교가 미리 등록한 질문과 답변으로 연결해주는 **지식+** 기능을 제공합니다
- 지식+에 등록 되어있지 않은 질문을 등록하는 **질문등록** 기능을 제공합니다

### 관리자 사이트 내 기능

- 학과별 학과공지, 학과별 교과과정, 학과별 사무실 정보, 수강편람, 학사 일정의 데이터를 변경 할 수 있습니다
- 교수님 데이터와 강의실 시간표 데이터를 추가,삭제,수정 할 수 있습니다
- 유저들이 궁금해하는 질문들을 **지식+** 질문으로 추가 하거나 미리 등록된 질문을 편집 할 수 있습니다
- 유저들이 가장 많이 질문하는 질문에 대한 데이터를 조회 할 수 있습니다
